Everactive Data Services Toolbox

The Everactive Data Services Platform lets you integrate the data collected and reported by Everactive sensors into your own industrial applications.

We can recommend some tools and open source projects that we love, to help you work with the platform. We have used these tools to develop samples and reference applications to assist you in integration with the Everactive Data Services Platform.

Data Services API Developer Documentation

The Everactive Data Services REST API gives you direct programmatic access to the data returned by Eversensors, as well as connectivity status and configuration information about the Eversensors and Evergateways in an Everactive installation.

  • See complete reference documentation here: Everactive API Reference

    • Data Services Reference Applications

    We have developed some projects to assist you in integration with the Everactive Data Services Platform.

  • dataservices-sample-client : Docker-based app to receive, store and review Eversensor time-series readings.
  • nodejs-webhook-sample : Sample Node.js project illustrates how to consume Everactive webhook messages.

    • Our reference applications make use of these open-source tools to provide development and production support.

  • Express, a Node.js development framework for web applications.
  • Docker Compose: A tool for defining and running multi-container Docker applications.
  • TimescaleDB: A relational database (a PostgreSQL extension) for time-series and analytics.
  • Grafana: A data visualization platform.
